O QUE É?

Uma GPU (Graphics Peocessing Unit), ou unidade de processamento gráfico, é um processador especializado nos calculos de geração de imagem, tendo seu início focado no processamento de imagens 2D e 3D, é usado hoje em dia para processamento de gráficos avançados em tempo real, como o Raytracing.



QUAL SUA UTILIDADE?


Processamento Gráfico

A GPU é principalmente usada hoje em dia para processamento gráfico em tempo real, ou seja, é principalmente usado para aprimorar elementos gráficos dinâmicos, como sombras, reflexos, etc.


JOGOS

A GPU também é usada para jogos digitais, a GPU faz o trabalho de renderização de elementos gráficos avançados em tempo real, como o raytracing, assim jogos com elementos gráficos avançados demandam uma GPU avançada para manter uma resposta de processamento em uma taxa de frames balanceados.


MODELAGEM

Modelagem 3D é algo muito presente nos tempos modernos, sendo muitos desses modelos de alta resolução, usando programas de renderização e modelagem em tempo real é preciso uma GPU boa para uma rápida renderização dos elementos, no caso do Blender ele é muito conhecido por ter entroduzido a renderização de sompras e reflexos em tempo real aos softwares de modelagem, e continua sendo usado até hoje.



ARQUITETURA

As GPUs são projetadas para processar multiplas vezes sequencialmente, por isso a GPU possui multiplos núcleos de processamento, assim fazendo a GPU ser útil para execução de qualquer tarefa que envolva processamento paralelo intenso.
